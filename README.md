## 🛠️ Installation

1. Clone repo:
   ```bash
   git clone https://github.com/yourusername/adventures-in-devoops.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start local server:
   ```bash
   npm run dev
   ```
4. Open `http://localhost:8080` in browser
